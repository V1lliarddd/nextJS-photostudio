## 🚀 Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm / yarn / pnpm

### Installation

1. **Clone the repository**

```
git clone https://github.com/yourusername/photostudio.git
cd photostudio
Install dependencies

npm install
# or
yarn install
# or
pnpm install
```

2. **Run the development server**

```
npm run dev
# or
yarn dev
# or
pnpm dev

Open http://localhost:3000 in your browser
```
